# advertisement-skills — Claude Code / Codex 广告技能包

本文件与 `README.md` 保持同一中文说明入口。

`advertisement-skills` 用于沉淀可复用的广告工作流，包括广告创意简报、媒介规划、创意审核、投放复盘和广告素材优化。

## 快速开始

```text
/plugin marketplace add saibodafu/advertisement-skills
/plugin install advertisement-skills
```

或在 Codex 中打开本仓库后直接使用：

```text
使用 ad-brief，帮我整理一份广告创意简报。
使用 media-plan，帮我做一版广告媒介规划。
使用 creative-review，帮我审一下这组广告创意。
使用 seedance-short-drama-storyboard，把这个短剧剧本拆成 Seedance 2.0 分镜提示词。
```

## 当前 Skills

- `ad-brief`：广告创意简报
- `media-plan`：广告媒介规划
- `creative-review`：广告创意审核
- `seedance-short-drama-storyboard`：Seedance 短剧 15 秒分镜
