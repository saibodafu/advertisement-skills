# 广告媒介规划

本技能用于把广告目标转化为可执行的媒介组合和投放节奏。重点不是生成漂亮的媒介理论，而是帮助用户知道应该投什么渠道、为什么投、怎么分工、怎么衡量。

## 使用场景

使用本技能处理以下需求：

- 做广告媒介规划
- 规划信息流、搜索、社媒、达人或本地生活广告组合
- 为新品上市、活动引流、销售获客制定投放节奏
- 拆解预算分配逻辑
- 梳理不同渠道的素材需求和数据指标

## 必要输入

尽量收集或推断以下信息：

- 产品、品牌或活动
- 广告目标
- 目标人群
- 投放地区和周期
- 预算范围
- 已有渠道偏好或限制
- 素材能力
- 转化承接方式

如果没有预算，可以用“低 / 中 / 高资源假设”给出规划，不要编造具体投放金额。

如果需要最新平台规则、竞价机制或广告产品信息，必须联网核验，并标注来源和时间。

## 工作流程

1. 判断目标类型：认知、兴趣、留资、成交、复购、召回或销售辅助。
2. 拆解人群路径：从发现、理解、信任、行动到复购，判断每个阶段需要的触点。
3. 选择渠道组合：说明每个渠道承担的角色，而不是只列平台名称。
4. 设计投放节奏：按预热、放量、优化、收口或复盘拆阶段。
5. 分配资源逻辑：说明预算、素材和人力如何倾斜。
6. 定义素材需求：明确每个渠道需要的素材版本和测试变量。
7. 定义衡量指标：按阶段输出核心指标和观察指标。

## 输出格式

```markdown
## 广告媒介规划

**投放目标：** ...
**目标人群：** ...
**周期：** ...
**预算假设：** ...
**转化承接：** ...

### 渠道组合
| 渠道 | 角色 | 适合原因 | 主要素材 | 核心指标 |
|---|---|---|---|---|

### 投放节奏
| 阶段 | 时间 | 目标 | 动作 | 判断标准 |
|---|---|---|---|---|

### 资源分配逻辑
...

### 素材测试计划
| 测试变量 | 版本方向 | 预期验证 | 后续动作 |
|---|---|---|---|

### 复盘指标
- 核心指标：...
- 过程指标：...
- 需要人工判断的信息：...

### 假设与风险
- ...
```

## 质量标准

- 每个渠道必须说明角色和选择理由。
- 投放节奏必须能指导实际排期。
- 指标要与广告目标对应，避免只堆 CTR、CVR、CPM 等术语。
- 不编造预算效果、平台 benchmark 或竞价成本。
- 如果用户给出预算和周期，要优先在该约束内做取舍。
